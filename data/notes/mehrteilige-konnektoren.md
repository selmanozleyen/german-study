# 🔀 Mehrteilige Konnektoren (zweiteilige)

> **Quelle:** deutschkurse bei der Universität München, Stufe 3: B2.1 (im Unterricht 18.6. eingeführt). Zweiteilige Konnektoren verbinden zwei Elemente — Satzteile oder ganze Sätze.

## Regel

| Konnektor | Bedeutung | Stellung / Hinweis |
|---|---|---|
| **entweder … oder** | das eine oder das andere — man muss sich entscheiden | *entweder* an Pos. 0/1 oder im Mittelfeld; *oder* immer an 0 |
| **nicht nur … sondern auch** | das eine reicht nicht, es kommt „extra" dazu | *nicht nur* meist im Mittelfeld; *sondern* im 2. Teil immer an 0 |
| **sowohl … als auch** | das eine und das andere | ⚠️ verbindet nur **Satzteile**, keine ganzen Sätze |
| **weder … noch** | das eine nicht, das andere auch nicht | schon **negativ** — kein zusätzliches nicht/kein |
| **zwar … aber** | Einräumung + Gegensatz | *aber* leitet den Gegensatz ein (oft + trotzdem) |
| **einerseits … andererseits** | zwei verschiedene Seiten / Aspekte | — |

## Beispiele

- **entweder … oder:** Entweder wir gehen ins Kino oder wir bleiben zu Hause. / Wir gehen **entweder** ins Kino **oder** wir bleiben zu Hause.
- **nicht nur … sondern auch:** Ich will **nicht nur** Bier trinken, **sondern auch** Billard spielen. / **Nicht nur** Erwachsene, **sondern auch** Jugendliche müssen streng bestraft werden. *(KB S. 207/A3)*
- **sowohl … als auch:** Ich kann **sowohl** Englisch **als auch** Deutsch sehr gut sprechen. / Wir können **sowohl** ins Kino **als auch** in die Kneipe gehen.
- **weder … noch:** Bei uns werden **weder** Körperverletzungen **noch** Sachbeschädigungen bestraft. / Ich will **weder** Bier trinken **noch** Billard spielen.
- **zwar … aber:** **Zwar** kommen heute gute Filme, **aber** ich gehe trotzdem gern mit dir in die Kneipe.
- **einerseits … andererseits:** **Einerseits** hast du kein Geld, **andererseits** gehst du dauernd aus.

## Stolperfallen

- **sowohl … als auch** verbindet nur Satzteile — *nicht* zwei ganze Sätze. (Für ganze Sätze: *nicht nur …, sondern auch …*.)
- **weder … noch** ist bereits verneint → **kein** zusätzliches *nicht*/*kein* einbauen.
- Nach einer Verneinung steht **sondern** (Korrektur), nicht *aber*: *nicht Bier, sondern Wein*.
- Im Nebensatz stehen die Teile vor den Alternativen: *Du weißt, dass ich nicht nur Bier trinken, sondern auch Billard spielen will.*
- Steigerung aus dem Unterricht: *sowohl … als auch* (+) ist neutral; *nicht nur … sondern auch* (++) ist betonter.
